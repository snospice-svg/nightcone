The original blue lotus run came back mostly null. The obvious objection: maybe the dose was too low. So this follow-up pushed it — roughly twice the typical amount, brewed as tea, across seven nights.

It didn't rescue the effect. Recall ticked up on a couple of nights, vividness was inconsistent, and there was no lucidity and no hypnagogic imagery at any point — the same shape as the first run, just louder in the cup, not in the dreams.

Dose-response is one of the cleaner things a single-subject protocol can probe: hold everything else fixed, change the amount, see if the signal scales. Here it didn't. That's stronger evidence for "this plant does little" than a single low-dose run alone.
